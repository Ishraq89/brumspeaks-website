# Brum Speaks Website

This repository contains the Brum Speaks landing page: index.html, style.css, script.js, and placeholder images.

To make the site live at https://www.brumspeaks.co.uk using GitHub Pages:

1. Ensure the CNAME file contains: www.brumspeaks.co.uk
2. In the repository settings -> Pages: select Branch: main and Folder: / (root).
3. Configure DNS: add a CNAME record for "www" pointing to ishraq89.github.io and optionally A records for the apex to GitHub Pages IPs.
